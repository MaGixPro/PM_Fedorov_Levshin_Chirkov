## 2) Экспертная оценка трудоёмкости
Метод: условный “Planning Poker” (учебная экспертная оценка) по шкале 1, 2, 3, 5, 8.

Каждый участник выставил оценки трудоёмкости для задач T1–T6.  
Для агрегирования использована PERT-формула на трёхточечной оценке:
- медиана как ожидаемая оценка,
- PERT = (min + 4*median + max) / 6.

Перевод в часы выполнен через коэффициент `k` (учебно), который задан в таблице.

## 4) Постановка задачи распределения
Переменная:
- x(t,p) ∈ {0,1} — задача t назначена участнику p.

Ограничения:
1) Каждая задача назначается ровно одному участнику:  Σ_p x(t,p) = 1
2) Каждый участник получает минимум одну задачу:          Σ_t x(t,p) ≥ 1
3) Ограничение нагрузки:                                  Load(p) ≤ LoadMax

Цель:
Сделать распределение “справедливым”: максимизировать минимальную суммарную предпочтительность среди участников (fairness):
- max z, где для каждого p выполняется  Σ_t pref(t,p)*x(t,p) ≥ z.
